Solution 9.3

It is clear that d(m, n) ≥ 0, that d(m, n) = 0 ⇔ m = n and that d(m, n) = d(n, m), so we need only verify the triangle inequality.
So suppose that 2r divides m - n and that 2s divides n - q. Then m - q = (m - n) + (n - q) and so if t = min{r, s} we have 2t divides n - q. But 1/2t≤ 1/2r+ 1/2s since t is either r or s. Hence the triangle inequality holds.

If an= 1/2n-1 then d(an, 0) = 1/2n-1 and so (an)→ 0 in this metric.

One can define similar metrics for any odd prime in a similar way. This gives rise to some interesting algebraic objects called the p-adic integers which arise in a wide variety of applications.
