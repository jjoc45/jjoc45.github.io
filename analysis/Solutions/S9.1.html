That is where -1/2 + 2/3a + b = 0 and -2/3 + a + 2b = 0 ⇒ 4a + 6b = 3, 3a + 6b = 2 ⇒ a = 1, b = -1/6. So the best-fit linear approximation to x2 is x - 1/6.
