Solution 1.5

The multiplicative identity is the "identity function": x ↦ x. It is a standard result in set theory that the invertible functions are those which are one-one and onto (bijections).
If x ∈ R then (f + g)h(x) = (f + g)(h(x)) = f(h(x)) + g(h(x)) = fh(x) + gh(x) = (fh + gh)(x).
If (say) h(x) = x2 then it is easy to verify that for most functions f, g the other distributive law fails.
Something like this is called a Near-ring.
